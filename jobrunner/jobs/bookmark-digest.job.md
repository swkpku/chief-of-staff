# Bookmark Digest

## Schedule
0 8 * * *

## Goal
Pull X bookmarks and create reminders to read them. Help process the bookmark backlog instead of letting it pile up.

## Policy
- Auto-set due date to 7 days after bookmark date
- After 14 days unread, move to deferred status
- Group by topic if possible
- Classify bookmarks by domain (career vs projects) based on content

## Boundaries
- Read-only access to bookmarks
- NEVER post tweets
- NEVER like or retweet
- NEVER delete bookmarks

## Tools
- x_bookmarks
